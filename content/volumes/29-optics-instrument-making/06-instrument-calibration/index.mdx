## Overview

A well-built telescope, microscope, or spectrometer is useless if you don't know if it's accurate. Calibration verifies that an instrument measures correctly.

Calibration without external standards is possible: compare the instrument to itself, use astronomical or natural constants, or construct a secondary standard from first principles.

## Prerequisites

- Understanding of all instrument types (Vol 32, Chs 1–5)
- Understanding of measurement and error (Vol 2, Ch 3)
- Access to natural constants or self-referential measurements
- Patience (calibration is slow and meticulous)

## Materials & Tools

- The instrument itself
- Reference objects (if available)

## Step-by-Step Instructions

### Part 1: Calibration Without External Standards

**Step 1: Use self-referential checks.**

Compare the instrument to itself:

**Telescope magnification check:**
1. Observe a distant object (church spire, mountain peak) at lowest magnification
2. Estimate its apparent size and distance
3. Switch to higher magnification
4. Re-estimate size and distance
5. Calculate magnification ratio: (low mag distance / low mag size) / (high mag distance / high mag size)
6. Compare to theoretical magnification (focal length of objective / focal length of eyepiece)

**Microscope magnification check:**
1. Observe a known specimen at 100x magnification
2. Estimate the size of a cell or feature
3. Switch to 400x magnification
4. The same feature should appear 4 times larger
5. Verify this ratio matches the magnification increase

**Spectrometer wavelength check:**
1. Observe known element spectra (hydrogen, helium from flames)
2. Measure the position of known lines (from literature or reference books)
3. Create a calibration curve: wavelength vs. position on your spectrometer
4. Use the curve to measure unknown spectra

**Step 2: Use natural constants.**

Several physical constants can verify instruments:

**Speed of light (for telescope + timing):**
1. Time how long starlight from Jupiter takes to reach Earth
2. Know Jupiter's distance (from ephemeris or astronomical tables)
3. Calculate: speed = distance / time
4. Should equal 300,000 km/s (or 3 × 10⁸ m/s)
5. Large error indicates a problem with the telescope or timing method

**Moon's angular size:**
1. Measure the Moon's apparent diameter through a telescope
2. Known value: about 0.5 degrees (same as the Sun, which is why solar eclipses work)
3. Calculate: actual diameter = (angular size in radians) × (distance to Moon)
4. Should be about 3,474 km
5. Verify against known values

**Stellar parallax (for telescopes):**
1. Observe a star's position at different times of year
2. Earth's orbit causes nearby stars to appear to shift position
3. Parallax = (shift / 2) in arcseconds
4. Distance (in parsecs) = 1 / parallax (in arcseconds)
5. Compare to known distances from other sources

**Step 3: Construct a secondary standard.**

If no external reference exists, build a reproducible standard:

**Mirror flatness verification:**
1. Compare your optical surface to another known flat (e.g., a very still water surface)
2. Set them side by side
3. If the water surface is flatter, your mirror needs more polish

**Lens focal length verification:**
1. Project the sun's image onto a screen
2. Measure the distance from the lens to the sharpest image
3. This distance is the focal length
4. Repeat with different screen distances to verify consistency

**Grating spacing verification:**
1. Diffraction gratings have known spacing (e.g., a CD's track spacing is about 1.6 μm)
2. Project laser light (if available) or sunlight through the grating
3. Measure the angle of diffraction
4. Calculate: wavelength = (grating spacing) × sin(angle)
5. Sunlight spectrum should show known wavelengths; verify your measurement matches

### Part 2: Identifying and Correcting Systematic Errors

**Step 1: Recognize systematic errors.**

**Zero error:** The instrument reads non-zero when it should read zero.
- Example: A telescope's cross-hairs are off-center
- Correction: Subtract the zero error from all measurements

**Scaling error:** The instrument reads too high or too low consistently.
- Example: A microscope's magnification is actually 150x, not the claimed 100x
- Correction: Multiply all measurements by the correction factor (true value / measured value)

**Non-linearity:** The error changes with the magnitude being measured.
- Example: A scale reads accurately at 100 g but poorly at 10 g or 500 g
- Correction: Create a calibration curve showing the relationship

**Environmental sensitivity:** Temperature, humidity, or vibration affects readings.
- Example: A telescope's focus drifts as temperature changes
- Correction: Account for environmental factors (take measurements in stable conditions)

**Step 2: Measure the error.**

1. Measure a known standard (or use self-referential measurements)
2. Compare the measured value to the true value
3. Calculate error: Error = (measured - true) / true × 100%
4. Repeat multiple times to identify systematic vs. random errors

**Step 3: Correct the error.**

**Additive correction (zero error):**
- Measured value = raw reading - zero error
- Example: If the instrument reads 10 mm when it should read 0, subtract 10 mm from all future readings

**Multiplicative correction (scaling error):**
- Measured value = raw reading × correction factor
- Example: If measurements are consistently 5% too high, multiply by 0.95

**Lookup table correction (non-linearity):**
- Create a table mapping raw readings to corrected values
- For any raw reading, look up the corresponding true value

### Part 3: Verification Tests

**Step 1: Test reproducibility.**

Measure the same object multiple times. Do you get the same result?

**Precision test:**
1. Measure a standard object at least 5 times
2. Calculate the average and standard deviation
3. Standard deviation should be small (under 1% for good instruments)
4. If large, the instrument is imprecise (high variability)

**Step 2: Test accuracy.**

Measure a known standard. Do you get the correct result?

**Accuracy test:**
1. Measure a known standard (from literature, known value, or trusted source)
2. Calculate error: (measured - true) / true × 100%
3. Error should be under 5% for typical post-collapse instruments
4. If larger, the instrument needs calibration adjustment or is faulty

**Step 3: Test linearity.**

Does the instrument measure correctly across its range?

**Linearity test:**
1. Measure several known standards (at low, medium, and high values)
2. Plot measured vs. true values
3. Should be a straight line through the origin
4. If curved, the instrument is non-linear and needs a calibration curve

### Part 4: Maintaining Calibration

**Step 1: Recalibrate periodically.**

Instruments drift over time due to:
- Mechanical wear
- Temperature changes
- Environmental contamination
- Age

**Recalibration schedule:**
- Frequently used instruments: monthly
- Occasionally used: quarterly
- Rarely used: before each use

**Step 2: Document calibration.**

Keep a log:
- Date of calibration
- Reference standard used
- Measured value and true value
- Correction applied
- Environmental conditions (temperature, humidity)

**Step 3: Store properly.**

Protect instruments from:
- Temperature extremes (causes expansion/contraction)
- Humidity (causes corrosion and optical fog)
- Vibration (causes mechanical stress)
- Dust (contaminates optics)

## Verification

A properly calibrated instrument:

- **Measures known standards correctly** (within 1–5% error)
- **Is reproducible** (repeated measurements are consistent)
- **Is linear** (error doesn't depend on the magnitude being measured)
- **Matches theory** (matches calculated values from first principles)

An uncalibrated or drifted instrument:

- **Measures known standards incorrectly** (error over 5%)
- **Is irreproducible** (repeated measurements vary widely)
- **Shows non-linearity** (error depends on the magnitude)
- **Conflicts with theory** (measurements don't match calculated values)

## Common Mistakes

- **Assuming accuracy without verification:** Just because an instrument looks well-built doesn't mean it's accurate. Always calibrate.
- **Ignoring systematic errors:** Repeatedly getting the same wrong answer is a problem. Correct systematic errors, not just random noise.
- **Using wrong standards:** If your reference is inaccurate, your calibration is wrong. Use the most accurate standards available.
- **Neglecting environmental factors:** Temperature and humidity affect many instruments. Calibrate in stable conditions.
- **One-time calibration:** Instruments drift. Recalibrate regularly.
- **Forgetting to document:** You won't remember what corrections were applied months later. Keep detailed records.

## Overview

Imagine explaining a water wheel or steam engine to someone who has never seen one. Oral instructions would be confusing and imprecise. A technical drawing shows every dimension, every angle, every material. Engineers use drawings to communicate with builders across time and distance.

A blueprint is a drawing with standardized conventions: lines represent edges, dimensions show sizes, symbols convey assembly instructions. Learning to read and create technical drawings is essential for post-collapse engineering.

## Prerequisites

- Understanding of mechanical concepts (Vol 33, Ch 1)
- Ability to measure precisely (Vol 2, Ch 3)
- Basic geometry (angles, proportions)
- Access to drawing materials (paper, pencil, ruler)

## Materials & Tools

- **Paper:** Drafting paper or parchment
- **Pencils:** Hard pencils (H or 2H) for light lines, softer (HB or B) for dark lines
- **Ruler:** Straight edge for drawing lines
- **Compass:** For drawing circles and arcs
- **Protractor:** For measuring angles
- **Eraser:** For corrections
- **Ink:** Optional, for final linework (more permanent than pencil)

## Step-by-Step Instructions

### Part 1: Understanding Technical Drawing Conventions

**Step 1: Learn line types and meanings.**

Different line styles convey different information:

**Visible (solid, thick) lines:**
- Edges you can see
- The outline of the object

**Hidden (dashed) lines:**
- Edges you cannot see (hidden behind other parts)
- Example: A hole drilled into the back of an object

**Center lines (dash-dot):**
- Mark the center of circles and symmetrical features
- Guide for the eye (not a real edge)

**Dimension lines (thin, with arrows):**
- Show measurements
- End with arrows pointing to the edges being measured

**Step 2: Understand views and projections.**

A 3D object is shown in multiple 2D views:

**Front view:**
- What you see looking directly at the front
- Shows height and width
- Usually the most important view

**Top view:**
- What you see looking down from above
- Shows width and depth

**Side view (profile):**
- What you see looking from the side
- Shows height and depth

**Isometric view (3D):**
- Shows the object in three dimensions
- Easier to visualize, but less precise than orthographic views

**Example:** A water wheel is drawn as:
1. Front view: Circle showing the wheel face-on
2. Top view: Oval showing the wheel from above (width and depth apparent)
3. Side view: The wheel's thickness and bucket depth shown

**Step 3: Learn dimensioning standards.**

Dimensions communicate exact sizes so builders can construct the object precisely.

**Dimension line format:**
- A thin line parallel to the edge being measured
- Arrows at the ends point to the edges
- A number above the line shows the measurement

**Placement:**
- Dimensions are placed outside the object (if possible)
- Dimensions for similar features are aligned in columns
- All dimensions are placed so the number reads horizontally or is written beside the line

**Units:**
- All dimensions must use the same units (metric is preferred post-collapse)
- Example: 25 mm, 1.5 meters, 0.5 mm

**Chaining dimensions:**
- Related dimensions are placed together
- Example: A shaft with segments: first 100 mm, then 50 mm, then 75 mm total length is 225 mm

### Part 2: Creating a Technical Drawing

**Step 1: Gather information about the object.**

Before drawing, know:
1. **Overall dimensions:** Length, width, height
2. **Key features:** Holes, slots, curves, steps
3. **Materials:** Steel, wood, iron
4. **Assembly:** How does this part connect to others?
5. **Function:** What does this part do?

**Step 2: Decide which views are needed.**

Not all objects need all three views:
- A simple flat plate: Front view and maybe top view
- A circular object (wheel): Front view and side view
- A complex part: All three views plus an isometric for clarity

**Step 3: Sketch lightly first.**

1. Use a light pencil (hard, like 2H or H)
2. Sketch the outline of each view
3. Don't worry about exact lines yet; just get the proportions right

**Step 4: Add details and features.**

1. Sketch any holes, slots, or curves
2. Add center lines for circles
3. Mark where dimensions will go

**Step 5: Add dimension lines and numbers.**

1. Measure or calculate exact dimensions
2. Draw dimension lines (thin, parallel to the edges)
3. Mark measurements clearly

**Step 6: Darken the final lines.**

1. Use a darker pencil (HB or B) for visible edges
2. Use light lines for hidden edges (dashed)
3. Ink over the pencil lines if creating a permanent blueprint

**Step 7: Add a title block.**

Include:
- Object name and description
- Scale (e.g., 1:10 means the drawing is 1/10 the actual size)
- Date created
- Creator's name
- Material
- Any notes or special instructions

### Part 3: Tolerances and Fits

**Step 1: Understand tolerances.**

No measurement can be perfect. Tolerances specify how much variation is acceptable.

**Examples:**
- Tight tolerance: 50.0 mm ±0.1 mm (must be between 49.9 and 50.1 mm)
- Loose tolerance: 50 mm ±1 mm (can be between 49 and 51 mm)

**Tight tolerances require:**
- More precise machining
- Careful measurement
- More time and skill
- More expensive

**Loose tolerances allow:**
- Hand tools and less precision
- Faster work
- Lower cost

**When to use tight tolerances:**
- Mating parts that must fit together precisely (gears, bearings)
- Parts that must align (cylinder head to block)

**When to use loose tolerances:**
- Non-critical dimensions (mounting holes if flexible)
- Hand-made parts where precision is difficult
- Cost-sensitive applications

**Step 2: Specify fits.**

A fit is how two parts relate to each other:

**Clearance fit:**
- The hole is larger than the shaft
- Parts slide together freely
- Example: A water wheel shaft in its bearing (slight clearance needed for lubrication)

**Interference fit:**
- The shaft is larger than the hole
- Parts must be pressed together
- Very tight connection
- Example: A gear on a shaft (must not slip under load)

**Transition fit:**
- Somewhere between clearance and interference
- Parts fit snugly but can still be assembled by hand

### Part 4: Assembly Drawings

**Step 1: Create an assembly drawing.**

An assembly drawing shows how multiple parts fit together.

**Contents:**
1. All component parts shown in their correct position
2. Each part numbered (balloons pointing to the parts)
3. A bill of materials (list of all parts with quantities)
4. Assembly notes (how to assemble, fasteners to use)

**Example: Water wheel assembly drawing**
1. Shows the wheel frame
2. Shows the shaft through the center (part 1)
3. Shows the buckets (parts 2–9, eight total)
4. Shows the bearings (parts 10–11, two total)
5. Bill of materials lists all parts and quantities
6. Assembly notes explain the sequence: Install shaft, install bearings, attach buckets, mount to frame

**Step 2: Use callouts and notes.**

**Callouts:**
- Arrows pointing to features with text labels
- Example: "Paint this surface" or "Drill and tap for 1/4-20 bolt"

**Notes:**
- Special instructions
- Example: "All surfaces to be finished smooth" or "Grind this surface flat to ±0.5 mm"
- Part list reference
- Example: "See detail A for connection method"

### Part 5: Common Drawing Mistakes and How to Avoid Them

**Step 1: Avoid ambiguity.**

Every feature should be clear:
- If a hole is drilled all the way through, note it (or show it in another view)
- If a dimension is critical, call it out explicitly (don't assume the builder will infer)
- If a feature is optional or conditional, state it clearly

**Step 2: Keep drawings readable.**

1. Use consistent line weights (thick, medium, light)
2. Place dimensions clearly outside the object
3. Avoid crossing dimension lines
4. Use standard abbreviations (not personal shorthand)

**Step 3: Provide sufficient views.**

1. Ensure a builder can construct the object from the drawing alone
2. If the drawing is ambiguous (a builder might make two different parts), add a view or note
3. Include cross-sections if the internal structure is important

**Step 4: Specify all necessary information.**

1. Materials (steel, wood, brass)
2. Finish (paint, polish, natural)
3. Fasteners (bolt size, quantity)
4. Tolerances (if precision is critical)
5. Assembly sequence (if parts must be assembled in a specific order)

## Verification

A good technical drawing:

- **Is unambiguous:** A builder could create the object without asking questions
- **Is to scale:** Proportions are accurate (or scale is clearly marked)
- **Is complete:** All necessary dimensions are shown
- **Is readable:** Lines are clear, text is legible
- **Follows conventions:** Uses standard symbols and formats

A poor drawing:

- **Is ambiguous:** Multiple interpretations are possible
- **Is out of scale:** Proportions are misleading
- **Is incomplete:** Missing dimensions for critical features
- **Is hard to read:** Cluttered, unclear, or illegible text
- **Uses non-standard notation:** Personal symbols that only the creator understands

## Common Mistakes

- **Missing dimensions:** A builder cannot construct the object without all dimensions. Include every measurement.
- **Inconsistent scale:** If different parts are drawn to different scales, mark each one clearly. Ambiguity causes errors.
- **Unclear views:** If the three views don't clearly show the 3D form, add an isometric view or cross-section.
- **Vague notes:** Notes like "machine smooth" are subjective. Instead, specify "grind to ±0.1 mm" or "hand-file smooth."
- **Wrong tolerance:** Specifying a tolerance that's impossible to achieve with available tools wastes time and money.
- **Incomplete assembly drawings:** If the drawing doesn't show how parts connect, the assembler must guess.
- **Ignoring fastener standards:** Using non-standard bolts or screws makes replacement difficult. Use standard sizes.

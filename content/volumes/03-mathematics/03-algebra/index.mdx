## Overview

Arithmetic works with known numbers. Algebra works with unknown numbers, represented as variables (usually letters like x, y, or z). By writing equations that describe relationships, algebra allows us to solve for unknown quantities.

This chapter covers the fundamental operations of algebra: variables, equations, solving, and practical application.

## Prerequisites

- Mastery of arithmetic: addition, subtraction, multiplication, division
- Understanding of negative numbers and fractions
- Ability to think abstractly about numbers

## Materials & Tools

### For learning
- Paper and pencil for writing equations and solving
- Objects to physically represent variables and equations (useful for building intuition)

### For application
- Understanding of how to set up equations from word problems
- Knowledge of checking solutions

## Part 1: Variables and Expressions

### What is a variable?

A variable is a symbol (usually a letter) that represents an unknown number. The letter stands in for a quantity we want to find.

**Example:** If you know that a number plus 5 equals 12, you can write:
- x + 5 = 12

Here, x is the variable (unknown number), and we want to find what x is.

### Algebraic expressions

An expression is a combination of variables, numbers, and operations (addition, subtraction, multiplication, division).

**Examples:**
- 2x + 3 (means: 2 times x, plus 3)
- 5y - 7 (means: 5 times y, minus 7)
- x² + x + 1 (means: x squared, plus x, plus 1)

### Simplifying expressions

Expressions can be simplified by combining like terms (terms with the same variable).

**Example: Simplify 3x + 2x + 5**

- Combine like terms: 3x + 2x = 5x
- Result: 5x + 5

**Example: Simplify 4y + 3 + 2y - 1**

- Combine y terms: 4y + 2y = 6y
- Combine number terms: 3 - 1 = 2
- Result: 6y + 2

### Evaluating expressions

Evaluating an expression means substituting a value for the variable and calculating the result.

**Example: Evaluate 3x + 2 when x = 4**

- Substitute 4 for x: 3(4) + 2
- Calculate: 12 + 2 = 14
- Result: 14

## Part 2: Equations and Solving

### What is an equation?

An equation states that two expressions are equal. It contains an equals sign (=).

**Example:**
- x + 5 = 12
- 2y - 3 = 7
- x² + x - 6 = 0

Solving an equation means finding the value (or values) of the variable that make the equation true.

### Solving one-step equations

**Type 1: Addition/Subtraction**

To isolate the variable, do the opposite operation on both sides.

**Example: Solve x + 5 = 12**

- Subtract 5 from both sides: x + 5 - 5 = 12 - 5
- Simplify: x = 7
- Check: 7 + 5 = 12 ✓

**Example: Solve y - 3 = 10**

- Add 3 to both sides: y - 3 + 3 = 10 + 3
- Simplify: y = 13
- Check: 13 - 3 = 10 ✓

**Type 2: Multiplication/Division**

Do the opposite operation on both sides.

**Example: Solve 3x = 12**

- Divide both sides by 3: 3x / 3 = 12 / 3
- Simplify: x = 4
- Check: 3(4) = 12 ✓

**Example: Solve y / 2 = 5**

- Multiply both sides by 2: (y / 2) * 2 = 5 * 2
- Simplify: y = 10
- Check: 10 / 2 = 5 ✓

### Solving multi-step equations

Combine the operations above, doing one step at a time.

**Example: Solve 2x + 3 = 11**

- Subtract 3 from both sides: 2x + 3 - 3 = 11 - 3
- Simplify: 2x = 8
- Divide both sides by 2: 2x / 2 = 8 / 2
- Simplify: x = 4
- Check: 2(4) + 3 = 8 + 3 = 11 ✓

**Example: Solve (y - 2) / 3 = 4**

- Multiply both sides by 3: ((y - 2) / 3) * 3 = 4 * 3
- Simplify: y - 2 = 12
- Add 2 to both sides: y - 2 + 2 = 12 + 2
- Simplify: y = 14
- Check: (14 - 2) / 3 = 12 / 3 = 4 ✓

### Solving equations with variables on both sides

**Example: Solve 2x + 3 = x + 7**

- Subtract x from both sides: 2x - x + 3 = x - x + 7
- Simplify: x + 3 = 7
- Subtract 3 from both sides: x = 4
- Check: 2(4) + 3 = 8 + 3 = 11, and 4 + 7 = 11 ✓

### Solving quadratic equations (second-degree)

Quadratic equations have a variable raised to the power of 2 (like x²).

**Quadratic formula:** For any equation of the form ax² + bx + c = 0, the solutions are:

```
x = (-b ± √(b² - 4ac)) / 2a
```

The ± means there can be two solutions (one using + and one using -).

**Example: Solve x² + x - 6 = 0**

- Identify a = 1, b = 1, c = -6
- Calculate b² - 4ac = 1² - 4(1)(-6) = 1 + 24 = 25
- Calculate √25 = 5
- Apply the formula: x = (-1 ± 5) / 2
- Solution 1: x = (-1 + 5) / 2 = 4 / 2 = 2
- Solution 2: x = (-1 - 5) / 2 = -6 / 2 = -3
- Check: 2² + 2 - 6 = 4 + 2 - 6 = 0 ✓ and (-3)² + (-3) - 6 = 9 - 3 - 6 = 0 ✓

## Part 3: Systems of Equations

A system of equations is a set of two or more equations with multiple variables. The solution is the set of values that satisfy all equations simultaneously.

### Solving by substitution

**Example: Solve the system**

```
x + y = 10
2x - y = 8
```

- From the first equation, solve for x: x = 10 - y
- Substitute into the second equation: 2(10 - y) - y = 8
- Simplify: 20 - 2y - y = 8
- Combine: 20 - 3y = 8
- Subtract 20: -3y = -12
- Divide by -3: y = 4
- Substitute back: x = 10 - 4 = 6
- Check: 6 + 4 = 10 ✓ and 2(6) - 4 = 12 - 4 = 8 ✓

### Solving by elimination

**Example: Solve the system**

```
3x + 2y = 13
2x - 2y = 2
```

- Add the two equations (the y terms cancel):
  ```
  3x + 2y = 13
+ 2x - 2y = 2
-----------
  5x + 0y = 15
  ```
- Solve: 5x = 15, so x = 3
- Substitute back: 3(3) + 2y = 13, so 9 + 2y = 13, so 2y = 4, so y = 2
- Check: 3(3) + 2(2) = 9 + 4 = 13 ✓ and 2(3) - 2(2) = 6 - 4 = 2 ✓

## Part 4: Practical Applications

### Setting up equations from word problems

**Example:** A farmer has chickens and goats. Chickens have 2 legs each, goats have 4 legs each. He has a total of 20 animals with 56 legs. How many chickens and goats does he have?

- Let c = number of chickens, g = number of goats
- Total animals: c + g = 20
- Total legs: 2c + 4g = 56
- Solve: From the first equation, c = 20 - g
- Substitute: 2(20 - g) + 4g = 56
- Simplify: 40 - 2g + 4g = 56
- Combine: 40 + 2g = 56
- Solve: 2g = 16, so g = 8
- Therefore: c = 20 - 8 = 12
- Check: 12 chickens + 8 goats = 20 animals ✓, and 12(2) + 8(4) = 24 + 32 = 56 legs ✓

### Calculating material needs

**Example:** You are building a fence. Posts cost 5 coins each, and sections cost 3 coins each. You need 10 more posts than sections and have a budget of 100 coins. How many of each can you buy?

- Let p = posts, s = sections
- Cost equation: 5p + 3s = 100
- Relationship: p = s + 10
- Substitute: 5(s + 10) + 3s = 100
- Simplify: 5s + 50 + 3s = 100
- Combine: 8s + 50 = 100
- Solve: 8s = 50, so s = 6.25

Since you can't buy a partial section, you would buy 6 sections and 16 posts, costing 5(16) + 3(6) = 80 + 18 = 98 coins.

## Verification

Mastery of algebra is demonstrated by:

1. **Simplifying expressions:** Correctly combining like terms.
2. **Solving equations:** Finding the correct value(s) for variables.
3. **Checking solutions:** Verifying that your answer is correct by substituting back.
4. **Setting up equations:** Translating word problems into algebraic equations.
5. **Solving systems:** Finding values that satisfy multiple equations simultaneously.

## Common Mistakes

1. **Forgetting to do the same operation to both sides:** If you add to one side, you must add to the other.
2. **Sign errors:** Paying attention to positive and negative signs, especially when multiplying or dividing.
3. **Not checking the solution:** Always verify by substituting back into the original equation.
4. **Incorrectly applying the quadratic formula:** Remember the ± (two possible answers) and the order of operations.
5. **Mixing up variables:** In systems of equations, keep track of which variable is which.
